Homework 6: Census data choropleth

Deadline:

hw5

Instructions: Make a choropleth with the county dataset of another state besides NC

Using census data that does not fall under the topics of “housing” or “population and people”. Videos 4h to 4j are the most related to this homework. The final map should have a state or UTM projection and should not be in a default projection like EPSG 4326 or in a global projection like 3857. The map does not need a scale or north (often choropleths do not benefit from a scale or north). However, the scale should be clear and should have text related to the variable being displayed. Do not leave the default names of the file to be displayed in the legend. The color gradient of the map should not be blue nor red (these colors are often related to political parties in the US). The map should have a title and your name. Upload to Canvas the map, the vector file (as a zip) and the CSV that you used for the join. Include a few sentences (as a document or in the Canvas text box) discussing the map and any interesting observation that you made. The vector file should only include the counties for the state that you selected, NOT the entire USA. Also upload the CSV that you used for the join. Ideally the CSV should be cleaned (ready to make a join) and not just the original file.

Upload the map, the csv, the description and the vector file to Canvas
